# redichat

실시간 레디스 터미널 채팅

```bash
npm install -g redichat
```

> Redis Pub/Sub을 이용한 간단한 실시간 다중 채팅방 터미널 애플리케이션입니다.

`redichat`은 Node.js, React (Ink), Redis로 만들어진 CLI 채팅 애플리케이션입니다.
터미널에서 직접 여러 사용자가 채팅방에 참여하고 실시간으로 소통할 수 있습니다.

## ✨ 주요 기능

- Redis Pub/Sub 기반의 **실시간 메시징**
- **다중 채팅방** 지원 (즉시 생성 및 참여 가능)
- 비밀번호로 보호되는 **비공개 채팅방**
- 채팅방 입장 시 **이전 대화 기록** 로드
- 사용자 간 **귓속말** 기능
- **다국어 지원** (한국어 / 영어) (예정)
- 가독성을 위한 **동적 닉네임 색상**
- Ink를 사용한 **터미널용 React**로 제작

# ⚙️ 설정

`redichat`을 사용하려면 접속할 Redis 서버가 필요합니다.
이 서버를 공유 디지털 공간 또는 '놀이터'라고 생각해보세요. **다른 사람과 대화하려면 모두가 같은 놀이터에 있어야 합니다.**

`redichat` 명령어를 실행하는 디렉터리에 `.env` 파일을 생성하여 어떤 놀이터에 접속할지 설정할 수 있습니다.

### 1. 친구들과의 비공개 채팅

친구들과 함께 사용할 비공개 공간을 만들고 싶다면, 모두가 **동일한 Redis 서버**에 접속해야 합니다.

1.  **서버 생성:** 그룹 중 한 명이 Redis Cloud나 Upstash 같은 서비스에서 무료 데이터베이스를 생성합니다.
2.  **접속 정보 얻기:** 서비스 대시보드에서 **호스트(Host)**, **포트(Port)**, **비밀번호(Password)**를 확인합니다.
3.  **안전하게 공유:** 이 정보를 친구들에게 비공개적으로 공유합니다.
4.  **`.env` 파일 생성:** 각자 `.env` 파일을 만들고 공유받은 접속 정보를 채워 넣습니다.

    ```env
    # .env - 비공개 그룹용 예시
    REDIS_HOST=redis-12345.c1.us-east-1-2.ec2.cloud.redislabs.com
    REDIS_PORT=12345
    REDIS_PASSWORD=공유받은_비밀번호
    ```

### 2. 개발자용 (로컬 테스트)

혼자서 개발하거나 테스트할 때는 로컬 컴퓨터에서 실행 중인 Redis 서버에 연결할 수 있습니다.

1.  저장소 클론: `git clone https://github.com/geunseonkim/redichat.git`
2.  의존성 설치: `npm install`
3.  예제 파일 복사: `cp .env.example .env`
4.  `.env` 파일의 기본값은 일반적으로 로컬 Redis 설정에 맞게 되어 있습니다.

    ```env
    # .env - 로컬 개발용 기본값
    REDIS_HOST=127.0.0.1
    REDIS_PORT=6379
    REDIS_PASSWORD=
    ```

### "random" 방에 대한 참고사항

방 이름으로 `random`을 입력하면, `redichat`은 **현재 연결된 Redis 서버**에서 공개된 방을 찾습니다. 이는 다음을 의미합니다.

- 만약 당신과 친구들이 **같은** Redis Cloud 서버에 연결되어 있다면, `random`을 입력했을 때 같은 공개 채팅방에 입장하게 됩니다.
- 만약 당신은 로컬 컴퓨터에, 친구는 친구의 로컬 컴퓨터에 연결되어 있다면, 각자 **분리되고 격리된** `random` 방에 들어가게 됩니다. 이 경우 서로 대화할 수 없습니다.

# 🚀 사용법

채팅 클라이언트를 시작합니다:

```bash
redichat
```

다음과 같은 안내가 표시됩니다:

1.  언어 선택
2.  닉네임 입력
3.  채팅방 참여 또는 생성

---

# 💬 명령어

채팅방 안에서 다음 명령어들을 사용할 수 있습니다:

```
/users
```

현재 방에 있는 사용자 목록을 봅니다.

```
/rooms
```

활성화된 모든 방과 참여 인원 수를 봅니다.

```
/join <방이름>
```

다른 방으로 이동합니다.

```
/whisper <닉네임> <메시지>
```

개인 메시지(귓속말)를 보냅니다.

```
/back
```

비밀번호 입력 화면에서 이전 화면으로 돌아갑니다.

# 🧰 개발

저장소를 클론합니다:

```bash
git clone https://github.com/geunseonkim/redichat.git
cd redichat
```

의존성을 설치합니다:

```bash
npm install
```

프로젝트를 빌드합니다:

```bash
npm run build
```

로컬에서 실행합니다:

```bash
npm start
```
